# Claude Code Instructions

Use `AGENTS.md` as the source of truth for this repository.

In short:

1. Install/start <https://github.com/easyeda/easyeda-api-skill> when live EasyEDA access is needed.
2. Run `npm.cmd install`.
3. Run `npm.cmd run fast`, `npm.cmd run pipeline`, and `npm.cmd run visual`.
4. Write back with `npm.cmd run apply:gated` only after all gates pass.
5. Never bypass the gate with low-level writer scripts for delivery.
